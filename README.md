## Development

### Local
```bash
yarn start
```
To run a local development server that serves the basic demo located in `demo/index.html`

### Testing
To run the suite of Web Test Runner tests, run
```bash
yarn test
```

### Storybook
To run a local instance of Storybook for your component, run
```bash
yarn storybook
```
